# Wear Are You -- GPS Bracelet

This is a senior design project by a group of students at Georgia Institute of Technology Fall 2016. 

The parts used in this project are the following (including parts to be used):
- GSM Click 3 (SIM800) http://www.mikroe.com/click/gsm3/
- GPS Click (ublox LEA-6S) http://www.mikroe.com/click/gps/
- Teensy 3.2 (Cortex-M4) https://www.pjrc.com/store/teensy32.html
- E-Ink Display (PervasiveDisplay) http://www.pervasivedisplays.com/products/144
- Other miscellaneous: GPS Antenna, GSM Antenna, Boost Converter (3.7V->5V)

The software/server used for this project are the following:
- Adafruit IO MQTT
- Google Map API (Code not included in this repository)
- Online Python (Code not included in this repository)
